# Rock Paper Scissors Game ✊📄✂️

A simple command-line Rock Paper Scissors game built using Python. The player competes against the computer, and the winner is determined based on classic game rules.

---

## Features

* Interactive gameplay
* Random computer choices
* ASCII art visuals
* Win/Lose/Draw detection
* User-friendly command-line interface

---

## Technologies Used

* Python 3
* Random Module

---

## Game Rules

| Choice   | Value |
| -------- | ----- |
| Rock     | 0     |
| Paper    | 1     |
| Scissors | 2     |

### Winning Conditions

* Rock beats Scissors
* Scissors beats Paper
* Paper beats Rock
